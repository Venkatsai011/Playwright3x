## 🗺️ Learning Path

| Chapter | What You Learn | Files |
|---------|---------------|-------|
| **01 — Basics** | Running JS, `console.log`, declaring variables with `let` | 2 |
| **02 — Identifiers** | Valid names, reserved words, `$` and `_` rules | 1 |
| **03 — Literals** | Every data type you can write inline — numbers, strings, booleans, objects, arrays, regex, null, undefined | 8 |
| **04 — Operators** | Type checking, assignment, math, comparisons, logical operators, and the `==` vs `===` trap | 6 |
| **Interview Notes** | In-depth concept breakdowns with ASCII pipelines, kitchen analogies, and interview-ready recaps | 3 |
